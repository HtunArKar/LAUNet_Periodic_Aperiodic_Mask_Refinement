# LAUNet_Periodic_Aperiodic_Mask_Refinement

To train the model, follow the train.py file, but you need to adjust with your training environment. But all the training workflow is the same, just environmental set up will vary dependion on your environment.

For the evaluation, use T-Whisper medium form Hugging Face, that is simple to use, instead of downloading the model wieght.

You can download the Gowajee-Emergency dataset here https://drive.google.com/drive/folders/1O-i0aURINoWwOVDTwxTMvyGwzxqzCo2h?usp=sharing
